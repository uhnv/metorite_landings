Meteorite Data Representation

The goal of this project is to demonstrate the ability to:

  • Consume data programmatically from a given resource
  • Understand the data presented
  • Organize/disseminate/communicate the data in a presentable format
  • Utilize a GitHub source code repository
The data is taken from NASA meteorite landings "Downloads & Resources" page.

In addition to requirements this web page is done using WSU Colors Guidance.

Consuming and Analyzing Data

Data analyzing was done in two parts of the project. In the navigation bar you can go to "Map visualization" or "Chart representation" to see different ways to consume provided data.

The "Map visualization" page is the representation of the data from JSON file. This is an interactive map with meteorite landings actross the World. To create this meteorite landings map I have used LeafletJS open source interactive map liblrary. Each marker on the map represents a meteorite landing indicating it's actual position on the map. The variety of the library functionality was used, you can click on the marker and see the pop-up label with additional information for particular meteorite. All data as the marker position and pop-up label is taken from the provided JSON file.
